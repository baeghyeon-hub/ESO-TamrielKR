# Destinations KR Minion Patch

이 패키지는 `Destinations` 원본 폴더에 덮어쓰는 한국어 패치입니다.

## 포함 파일

- `Destinations/data/kr/DestinationsSettings_kr.lua`
- `Destinations/data/kr/DestinationsCollectibles_kr.lua`

## 설치 방식

압축본 또는 이 폴더 안의 `Destinations` 폴더를 원본 애드온 폴더 위에 그대로 덮어쓰면 됩니다.

## 메모

- 원본 매니페스트의 `data/$(language)/...` 구조를 그대로 사용합니다.
- 애드온 목록 제목 `Destinations`는 영문으로 유지하고, 설정 UI/필터/POI 타입 위주로 한국어를 덮어씁니다.
- `DestinationsCollectibles_kr.lua`는 영어 기본 데이터를 유지하는 빈 오버라이드 파일입니다.
