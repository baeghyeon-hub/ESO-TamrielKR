TamrielTradeCentre - Korean Patch

Standalone Korean patch addon for TamrielTradeCentre.

Installation:
1. Install the original TamrielTradeCentre addon first.
2. Extract this patch zip into your ESO AddOns folder as a separate addon.
3. Enable both TamrielTradeCentre and TamrielTradeCentre - Korean Patch in game.

Notes:
- This patch is intended for Korean TamrielKR users.
- It adds Korean TTC language data and the Korean item lookup table.
- It does not replace the original TamrielTradeCentre addon.

Required addon:
- TamrielTradeCentre

Optional addon:
- TamrielKR
