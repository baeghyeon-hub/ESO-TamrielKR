# HarvestMap KR Minion Patch

Adds a Korean localization overlay for `HarvestMap`.

Included files:
- `HarvestMap/Modules/HarvestMap/Localization/kr.lua`

Install:
- Merge the packaged `HarvestMap` folder into your existing `HarvestMap` addon folder.

Notes:
- This patch keeps the addon name `HarvestMap` in English.
- The visible settings/menu strings are translated through the addon's native `Localization/$(language).lua` hook.
